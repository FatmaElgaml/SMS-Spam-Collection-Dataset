df.columns

مثال:



\['v1', 'v2', 'Unnamed: 2', 'Unnamed: 3']

2\) .str.contains('^Unnamed')

df.columns.str.contains('^Unnamed')

معناها:



👉 افحص كل اسم عمود:

هل يبدأ بكلمة "Unnamed"؟



📌 معنى ^Unnamed

^ في regex معناها:

👉 "يبدأ بـ"



يعني:



^Unnamed = أي اسم يبدأ بـ Unnamed

📌 الناتج بيكون Boolean mask:

\[False, False, True, True]



يعني:



False → العمود مهم

True → العمود Unnamed

3\) علامة \~

\~df.columns.str.contains('^Unnamed')

معناها:



👉 اقلب القيم



قبل	بعد

True	False

False	True

بعد القلب:

\[True, True, False, False]

4\) .loc\[:, ...]

df.loc\[:, mask]

معناها:

: → خُد كل الصفوف

mask → اختار الأعمدة اللي True فقط

***text = "I have 10 apples and 5 oranges"***

&#x09;	***\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\****

***🔹 search***

***re.search(r'\\d+', text).group()***

***# '10'***

&#x09;	***\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\*\****

***🔹 sub***

***re.sub(r'\\d+', 'X', text)***

***🔹 split***

***re.split(r'\\s+', text)***

***search  → أول نتيجة***

***sub     → استبدال***

***split   → تقسيم***
